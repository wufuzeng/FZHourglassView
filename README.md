## 安装

FZHourglassView 可通过[CocoaPods](https://cocoapods.org)获得. 要安装它，只需将以下行添加到Podfile文件

```ruby
pod 'FZHourglassView'
```

## 怎样使用

* Objective-C

```objective-c
 
FZHourglassView *hourglassView = [FZHourglassView new];
[self.view addSubview:hourglassView];
hourglassView.center = self.view.center;
[hourglassView show];

```
